# "Kamsaw" hacker portfolio — offensive security architect

Single-page cyberpunk/hacker portfolio. The site showcases certifications (OSCP, CEH Master, CPTS, CWEE, AWS/GCP/Azure Architect, CKA, CKAD, CKS), the experience of an offensive cloud specialist, and interactive modules (terminal feed, skill cloud, animated contact cards) on top of a matrix rain backdrop.

## Local preview

1. Open `index.html` in your browser.
2. For full visual effects enable hardware acceleration in the browser.

## Structure

- `index.html` – entire application including styles and scripts.
